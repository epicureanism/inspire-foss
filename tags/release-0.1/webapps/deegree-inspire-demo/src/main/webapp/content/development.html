Development

Development of this demo was started by the deegree Team (Markus Schneider, Andreas Schmitz) and Just van den Broecke during the OSGeo Bolsena Code Sprint 2010. Here are also slides from a presentation in Bolsena.

This demo is part of the project, inspire-foss, that aims to provide Free and Open Source (FOSS) solutions for INSPIRE data transformation, storage and delivery (WFS/WMS). You are welcome to join this project (see Contact)

The "inspire-foss" project is hosted at: code.google.com/p/inspire-foss (opens in new window)

The Map and in fact this entire website was made with a separate FOSS project: code.google.com/p/geoext-viewer (opens in new window)
