Integration with local/national data sets

You may integrate local/national geodata as part of an ETL process. The ETL "Load" step may be performed as follows.

  • Insert INSPIRE data via WFS-T (see above)
  • Insert INSPIRE data directly in a database (e.g. PostGIS)
  • Insert INSPIRE data via custom deegree Feature Store Client
