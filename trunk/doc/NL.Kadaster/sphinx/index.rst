Home
====

This is the documentation for the project "INSPIRE Prototypes (phase2)"
performed within the GEORZ-lab & Research department of the Dutch Kadaster
(Eng: Cadastre, Land Registry and Mapping Agency) .
The home page for this project is http://inspire.kademo.nl.

*Note: This describes the second, most current, phase of INSPIRE prototyping.
For the, older, first phase (Phase 1) see* http://inspire.kademo.nl/docold/phase1
