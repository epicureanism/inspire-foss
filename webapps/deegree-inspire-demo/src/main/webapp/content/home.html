About

This is a demonstration of INSPIRE feature data storage and delivery using deegree 3 OGC web services. It showcases INSPIRE Download and View Services via WFS and WMS for Annex I Data Themes.


How to use this demo

This demo provides deegree 3 WFS and WMS services. The deegree 3 WFS is configured to serve all INSPIRE Annex I schemas. Feel free to try to insert any valid INSPIRE feature collection using a WFS transaction (WFS-T) and query it afterwards using the Client. The Map shows the same INSPIRE data as WMS layers. For the following INSPIRE Data Themes, some features are already inserted and have corresponding layers in the deegree 3 WMS:

  • Addresses (AD)
  • Cadastral Parcels (CP) - todo
