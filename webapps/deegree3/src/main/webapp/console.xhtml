deegree 3 services

Please note that this web interface is still in beta stage -- it's not a stable component of the deegree 3.0 release.

Currently, not all aspects of the service configuration can be configured using this interface and sometimes things may not work as expected. There are known issues with Chrome and Safari browsers, it's best tested with Firefox. For information on manual configuration of deegree 3, please refer to the deegree wiki.

That said, enjoy and let us know if you encounter issues. Also, you may want to consider contributing to improve it.
